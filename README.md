# homebrew-tap

Personal Homebrew tap for [vietairs](https://github.com/vietairs) tools.

```bash
brew tap vietairs/tap
brew install hvn-agentkit   # hak (Hvn-AgentKit) installer CLI
```

Note: `hak` installs skills/agents/hooks content from a private repo — your own
`gh`/git credentials need read access to `vietairs/hvn-agentkit` at runtime.
